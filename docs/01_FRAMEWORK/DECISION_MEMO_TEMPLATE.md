# decision_memo

## purpose
- slow down decisions
- reduce noise
- ensure one clear step

this is a thinking container.  
not a plan.

---

## ∴ node
- name the core decision
- one sentence only
- no context yet

---

## ⧖ context
- 3 facts max
- no interpretations
- no emotions

fact ≠ assumption

---

## ⧖ friction check
- internal resistance?
- external pressure?
- time distortion?

if pressure detected:
⧖ pause  
do not proceed

---

## ∴ criteria
- what makes this decision good?
- one criterion only
- observable

---

## ⟁ step
- one concrete action
- ≤10 minutes
- possible today
- reversible

no preparation steps.  
no chains.

---

## ➤ options (only if needed)
➤ option a  
➤ option b  
➤ option c  

no ranking  
no recommendation  

---

## 〄 closure
- decision fixed
- next review date (if needed)
- no expansion

---

## validation
- one node?
- one step?
- no pressure?
- canon respected?

if any fail:
↺ rebuild  
or ∅ silence
