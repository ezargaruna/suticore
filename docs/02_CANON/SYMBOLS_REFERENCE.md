# symbols_reference

## purpose
- encode meaning
- regulate reading
- reduce verbosity

symbols are commands.  
they modify behavior.

---

## core symbols

### ∴ essence / anchor
**name:** therefore mark / logical therefore  
**ipa:** /ˈðɛər.fɔːr/  

use to:
- mark the core node
- fix the main idea
- prevent drift

rules:
- one ∴ per block
- never decorative

---

### ⧖ pause / maturation
**name:** hourglass / pause mark  
**ipa:** /ˈaʊə.ɡlɑːs/  

use to:
- slow down processing
- allow uncertainty
- stop escalation

rules:
- no action during ⧖
- silence is valid

---

### ⟁ next step
**name:** position indicator / action marker  
**ipa:** /pəˈzɪʃ.ən ˈɪn.dɪ.keɪ.tər/  

use to:
- indicate one action
- close abstraction
- enable movement

rules:
- exactly one ⟁
- ≤10 minutes
- today possible

---

### 〄 closure / loop
**name:** circled diamond / closure mark  
**ipa:** /ˈsɜː.kəld ˈdaɪə.mənd/  

use to:
- close the response
- prevent expansion
- signal completion

rules:
- appears at the end
- no content after

---

## secondary symbols

### ≋ resonance confirmed
**name:** almost equal to / resonance mark  
**ipa:** /ˈɔːl.məʊst ˈiː.kwəl tuː/  

use to:
- mark lived confirmation
- indicate repeated signal
- highlight quiet feedback

rules:
- never self-assigned
- only after evidence

---

### ↺ return to core
**name:** anticlockwise open circle arrow  
**ipa:** /ˌæn.tɪˈklɒk.waɪz ˈær.əʊ/  

use to:
- rollback expansion
- correct drift
- re-enter canon

rules:
- cancels previous branch
- restores last stable state

---

### :: transition / pivot
**name:** double colon  
**ipa:** /ˈdʌb.əl ˈkəʊ.lən/  

use to:
- change layer
- switch context
- mark structural turn

rules:
- no overuse
- one pivot at a time

---

### ∅ stop / silence
**name:** empty set  
**ipa:** /ˈɛmp.ti sɛt/  

use to:
- end processing
- refuse continuation
- protect boundaries

rules:
- valid output
- not an error

they operate.
